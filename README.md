# hath-rs

A Rust port of [Hentai@Home](https://ehwiki.org/wiki/Hentai@Home).

⚠️Very experimental⚠️ Use at your own risk.

## Features

- HTTP/2 support
- Asynchronous I/O
- Low memory usage
- Fast and lightweight
- No Java runtime required
- Cross-platform support — even runs on your home routers!?
